Проект посвящён комплексному анализу базы данных вакансий с сайта **HeadHunter** (hh.ru). Исследование охватывает общую статистику рынка труда, детальный анализ вакансий по регионам, работодателям, сферам деятельности, требованиям к опыту и ключевым навыкам. Особое внимание уделено вакансиям в области **Data Science**.

## Цель проекта

Цель — провести разведочный анализ данных (EDA) и извлечь полезную информацию о рынке труда на основе реальных вакансий HeadHunter. Задачи включали:

- Определение объёма данных и предварительный анализ.
- Изучение распределения вакансий по регионам, работодателям и отраслям.
- Анализ зарплатных ожиданий в зависимости от опыта.
- Выявление ключевых навыков, востребованных в Data Science.
- Построение интерактивной географической визуализации зарплат и количества вакансий по городам-миллионникам России.

## Используемые данные

Данные хранятся в базе **PostgreSQL** и включают следующие таблицы:

- `vacancies` — информация о вакансиях (название, зарплата, опыт, регион и т.д.)
- `employers` — работодатели
- `areas` — регионы и города
- `industries` — сферы деятельности
- `employers_industries` — связь работодателей и сфер деятельности

Общий объём выборки:

- **49 197** вакансий
- **23 501** работодатель
- **1 362** региона
- **294** отрасли

- **Python 3** — визуализация, парсинг, координация.
